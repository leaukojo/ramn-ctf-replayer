Reset RAMN Board to DFU Mode

Use this tool to put ECU A into DFU (Device Firmware Upgrade) mode so you can reflash standard firmware with ramn-flasher.

This will open a Web Serial connection and send the DFU command automatically. If the board is running ESV CTF firmware, the selfdestruct command will be used instead.

Unofficial tool, not affiliated with the RAMN team. Based on publicly available data from the RAMN repository and official write-ups. Use at your own risk. Created with Claude Sonnet 4.6.
